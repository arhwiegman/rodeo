Initialization of components

Fortran 95 allows us to specify that any object of a derived type is given a default initial value for a component. The value is specified when the component is declared as part of the type definition. If the component is not a pointer, this is done in the usual way. If the component is a pointer, the only initialization allowed is to NULL(). Initialization does not have to apply to all components of a given derived type:

 TYPE node
    REAL                :: value = 0.0
    INTEGER                index
    TYPE(node), POINTER :: next => NULL()
 END TYPE node
